# Hacker rank

## Solve me first

Run:

    echo -e "2\n3" | cargo run --bin solve-me-first

Expected Output:

    5

## Solve me second

Run:

    echo -e "2\n2 3\n3 7" | cargo run --bin solve-me-second

Expected Output:

    5
    10

## Flipping bits

Run:

    echo -e "3\n2147483647\n1\n0" | cargo run --bin flipping-bits

Expected Output:

    2147483648
    4294967294
    4294967295

## Find digits

Run:

    echo -e "2\n12\n1012" | cargo run --bin find-digits
